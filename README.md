# 基于相对逆序数原理大规模榜单合成

## 一、本文完成的工作

1. 完成了基于相对逆序数原理的，适用于大规模榜单排序合成的工程实践
2. 优化论文原有流程，对其进行增补和改进
3. 对组件原理展开讨论，对组件参数进行调研
4. 进行消融实验，对算法功能进行讨论

## 二、数据
随机生成2000高校，11个榜单排名数据，见output文件夹。

随机数管线方面：
- 为模拟学校画像，选取research、teaching、international、reputation四维度，叠加异方差噪声， 顶尖学校噪声更小，弱校噪声更大
- 基于Beta分布生成潜在质量再进行幂变换，让头部学校差距拉开，尾部被轻微压缩。
- 为每套榜单的不同侧重点生成不同随机权重，有固定类型+Dirichlet扰动模拟现实世界。

## 四、本文贡献算法改进和创新点

1. 引入SCC预处理分组以分治问题，SCC无需基于有序数据产生。
2. 引入蒙特卡洛模拟退火解决 Condorcet 循环密集区的优化问题
3. 用滑动窗口（加入步长概念）优化原论文强制分段穷举法。
4. 将算法适配到大规模问题（适用规模从n≈20 扩展到 n≈2000，提升了两个数量级。复杂度从20!到2000!），数据集从"专家评审"场景跨越到"排名聚合"场景。
5. 提出了"逐级缩小误差尺度"的多级流水线架构。

| 步骤 | gap 到下界 | 作用 |
|------|-----------|------|
| 随机基线 | 703.3% | — |
| + Borda | 3.047% | 消除百位级误差 |
| + 插入排序 | 2.782% | 消除十位级误差 |
| + 模拟退火 | 2.255% | 跳出局部最优 |
| + 滑动窗口 | 2.246% | 精确微调 |

6. 蒙特卡洛模拟退火是本文重要贡献，后续可以计算其最优参数（随机调度跨度，随机调度模式，迭代次数与性能提升曲线等）
